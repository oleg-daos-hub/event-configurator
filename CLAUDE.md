# Event Configurator

Одностраничный конфигуратор и калькулятор корпоративных ивентов.

## Стек
Vanilla HTML/CSS/JS — никаких фреймворков, никакого build-шага. Открывается прямо в браузере.

## Структура
- `index.html` — вёрстка (конфигуратор + checkout на одной странице)
- `styles.css` — все стили
- `script.js` — вся логика и данные; цены — константы в начале файла
- `vercel.json` — настройки деплоя

## Принципы
- Не добавлять зависимости без явной просьбы
- Не усложнять — минимум кода для задачи
- Деплой через Vercel (push в main = автодеплой)
